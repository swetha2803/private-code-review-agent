Quick Start

Use this flow if you are opening the tool for the first time inside Citrix.

1 Add code

Upload a ZIP, upload a folder, upload files, or paste one file.

2 Run review

Click Run Review to generate findings, API list, flow, and risk summary.

3 Fill context

Add FS, LLD, POCs, issues, EA/ISG evidence, and review profile.

4 Export outputs

Export tracker CSV, mails, review pack, EA/ISG pack, or session JSON.

Citrix mode: Everything in this browser page runs locally. Do not paste real customer data, passwords, OTPs, tokens, or unmasked account/card numbers.
